# Campus Notification App

A small notification application with a Next.js frontend and notification design notes.

## What it is
- Frontend built with Next.js 14, React 18, and MUI.
- Supports notification listing, priority filtering, and read state.
- Includes API design ideas for student notifications and realtime updates.

## Key files
- `notification_system_design.md` — API summary and data model notes.
- `notification_app_fe/` — Next.js frontend source.
- `notification_app_fe/src/app/api/notifications/route.js` — example notifications API route.
- `notification_app_fe/src/components/NotificationCard.jsx` — UI card for notifications.

## Run locally
1. Go to `notification_app_fe`
2. Install dependencies: `npm install`
3. Start dev server: `npm run dev`

## Notes
- The frontend is the main working app in this workspace.
- Design doc captures notification API behavior and realtime update ideas.
